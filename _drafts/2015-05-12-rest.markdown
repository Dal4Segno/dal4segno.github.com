프로그램 혹은 서비스, 특히 웹 서비스를 개발하다 보면 REST API, RESTful 서비스와 같은 단어를 어렵지않게 볼 수 있다. REST란 **Re**presentational **S**tate **T**ransfer의 약자로, 다음과 같은 특징을 갖는 프로토콜이다.

 - 무상태성(Stateless)
 - 클라이언트/서버(Client/Server) 모델
 - 캐싱가능한 통신(Cacheble Communication)
